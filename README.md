# 🧠🎈 MomDom.

MomDom. is a family management platform designed to reduce the mental load for moms by centralizing daily life tracking, planning, and execution in one system.

~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*

## MVP1 Scope

This repository supports the MVP1 build of MomDom., focused on core functionality:

### Core Systems

* Family structure (families, users, members, relationships)
* Daily logging (daily entries, entry items)
* Task management (simple task tracking)
* Event scheduling (household events)
* Health tracking (health records, allergies, growth logs)
* Meal tracking (meals, meal entries)

~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*

## Product Philosophy

MomDom prioritizes:
* Low friction input
* Real-life usability
* Memory + execution support (not over-engineered systems)

This MVP intentionally avoids complex features (recurring logic, deep health modeling, recipe engines) to stay build-focused.

~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*~*

## Data Structure

The full MVP1 schema has been defined and locked in Google Sheets.

Key tables include:

* families
* users
* family_members
* tasks
* household_events
* health_records
* allergies
* meals
* meal_entries
